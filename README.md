# WETUBE CLONE

Wetube Clone built using NodeJS, Express, MongoDB and ES6

- rootRouter
  - home "/"
  - login "/login"
  - join "/join"
- videoRouter
  - upload video "/video/upload"
  - video "/video/:id"
  - edit video "/video/:id/edit"
  - delete video "/video/:id/delete"
- userRouter
  - user detail "/user/:id"
  - edit user "/user/:id/edit"
  - logout user "/user/:id/logout"
